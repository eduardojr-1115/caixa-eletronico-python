# 🏦 Caixa Eletrônico Chama Violeta 🔮

Projeto simples em Python simulando um caixa eletrônico.

## 🚀 Funcionalidades
- Saque com validação
- Cálculo de notas (200, 100, 50, 20, 10)
- Verificação de saldo
- Interface simples no terminal

## 💻 Tecnologias
- Python
